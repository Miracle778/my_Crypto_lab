
# 凯撒加密go语言实现

# 设计过程

分为三个模块

## 1.输入交互模块
先选择功能 加密或解密
选择好后，输入明文/密文，然后密钥
把输入的数据传给加解密模块

定义一个结构体作为该模块的传递消息格式
```go
type Msg struct {
    Flag int // 0代表加密 1代表解密
    Data string //明文或密文
    Key int  //密钥
}
```
密文和明文可以含有空格，于是Msg.Data用bufio.NewReader()进行输人
密文明文里也可以含有数字，数字加解密以10为一轮

## 2.Caesar密码算法模块
根据输入模块传进来的参数，进行Caesar算法加/解密，然后把结果传给输出模块

## 3.输出结果
把加/解密结果输出
